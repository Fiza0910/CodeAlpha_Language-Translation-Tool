A simple web application that allows users to translate text from one language to another using the Microsoft Translator API. Users can enter text, choose the source and target languages, and instantly view the translated result.

FEATURES

- Translate text between multiple languages
- Select source and target languages
- Fast and accurate translations
- Clean and user-friendly interface
- Displays translated text instantly
- Copy translated text (if implemented)
- Text-to-Speech support (if implemented)

TECHNOLOGIES USED

- HTML
- CSS
- JavaScript
- Microsoft Translator Text API

HOW IT WORKS

1. Enter the text you want to translate.
2. Select the source language.
3. Select the target language.
4. Click the **Translate** button.
5. The translated text is displayed on the screen.

HOW TO RUN ON YOUR DEVICE

1. Clone this repository

```bash
git clone https://github.com/your-username/Language-Translation-Tool.git
```

2. Navigate to the project folder

```bash
cd Language-Translation-Tool
```

3. Add your DeepL API credentials in the appropriate configuration file or `.env` file.

4. Open `index.html` in your browser or run the project using a local server.
